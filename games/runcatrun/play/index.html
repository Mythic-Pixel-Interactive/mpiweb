Description

Run Cat Run is a fast-paced, endless runner game where you take control of an agile cat on a daring dash through the forest! Leap over trees, avoid obstacles, and see how far you can go. With simple tap-to-jump controls, arcade-like visuals, and addictive gameplay, Run Cat Run is perfect for players of all ages looking for a fun and challenging experience on the go. How far can your cat run?
