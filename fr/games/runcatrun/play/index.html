Description

Run Cat Run est un jeu de course effréné et sans fin où vous incarnez un chat agile et partez à la conquête de la forêt ! Sautez par-dessus les arbres, évitez les obstacles et découvrez jusqu'où vous pouvez aller. Avec des commandes simples à utiliser, des graphismes dignes d'une arcade et un gameplay addictif, Run Cat Run est parfait pour les joueurs de tous âges en quête d'une expérience amusante et stimulante en déplacement. Jusqu'où votre chat peut-il courir ?
